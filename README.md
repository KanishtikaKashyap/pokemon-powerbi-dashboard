# pokemon-powerbi-dashboard
Built an interactive dashboard to analyze Pokémon statistics, type distributions, and performance metrics. This project demonstrates data transformation, DAX calculations, KPI tracking, and advanced visualization techniques in Power BI.  Skills: Power BI • Power Query • DAX • Data Modeling • Data Visualization • Business Intelligence

# Objectives
Analyze Pokémon attributes and characteristics.
Compare Pokémon performance metrics.
Identify top-performing Pokémon.
Explore distribution across different Pokémon types.
Create an interactive dashboard for data-driven insights.

# Tools Used
Power BI
Power Query
DAX
Data Visualization

# Dashboard Features
Interactive Filters and Slicers
Type-wise Analysis
Performance Comparison
Summary KPI Cards
Dynamic Visualizations
User-Friendly Dashboard Design

# Key Insights
Distribution of Pokémon across different types.
Strongest Pokémon based on overall statistics.
Comparison of Attack, Defense, Speed, and HP metrics.
Identification of top-performing categories.
